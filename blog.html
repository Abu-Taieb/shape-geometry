1st Question :: What is DOM? What is the purpose of DOM?

Answer :: DOM means Document Object Model. We are use this for change any element from the web pages. We can also add events to these elements to make our page more dynamic. It is a programming interface that allows us to create, change, or remove elements from the web page.

2nd Question :: How to select HTML element using by DOM?

Answer :: We can select HTML element using by DOM from our document in different ways through DOM

1. document.getElementById

2. document.getElementsByClassName

3. document.getElementsByName

4. document.getElementsByTagName

5. document.querySelector

6. document.querySelectorAll

4th Question :: What is callback function? Why we can use this?

Answer :: The callback function is the function we can use repeatedly in our document. Any function that is passed as an argument to another function so that it can be executed in that other function is called a callback function. We can create a function in a specific place and use that function repeatedly.
